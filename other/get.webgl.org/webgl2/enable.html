Enabling WebGL2

As of September 13, 2016 WebGL2 is still an experimental web technology. It is available in both Firefox 48 and Chrome 53.

To enable it:

  • In Firefox
    1. Go to about:config
    2. search for webgl2
    3. double click on webgl.enable-prototype-webgl2 until the value is true
    4. restart Firefox
  • In Chrome
    1. Go to about:flags
    2. Press Ctrl-F or Cmd-F and search for webgl 2.0
    3. Find WebGL 2.0 Prototype and click enable
    4. restart Chrome
