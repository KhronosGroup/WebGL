The files in this directory are referenced by the WebGL conformance tests:
  https://github.com/KhronosGroup/WebGL
which are hosted at:
  https://www.khronos.org/registry/webgl/sdk/tests/webgl-conformance-tests.html
and earlier snapshots at:
  https://www.khronos.org/registry/webgl/conformance-suites/

The files are hosted on opengl.org in order to test the WebGL API's
behavior with cross-origin resources.

and opengl_logo.jpg originally came from:
